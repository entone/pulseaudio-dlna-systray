####System tray support for [pulseaudio-dlna](https://github.com/masmu/pulseaudio-dlna).

You don't need to run pulseaudio_dlna.py. `systray.py` loading module-dbus-protocol and run pulseaudio_dlna.py for you. You can control from systray menu.

*Install*: put `systray.py` & `systray.png` to root directory of [pulseaudio-dlna](https://github.com/masmu/pulseaudio-dlna).

*Run*: `systray.py &`
